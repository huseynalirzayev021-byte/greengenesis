### About the Project

GreenGenesis is a student-led environmental awareness platform dedicated to protecting Azerbaijan's environment. The platform educates users about ecological facts, helps calculate carbon footprint, offers a rewards program for planting trees and plants, and facilitates donations for environmental projects.

### Key Features

- **Home Page**: Environmental facts and statistics about Azerbaijan
- **Awareness**: Carbon footprint calculator
- **GreenRewards**: Rewards program with AI-powered receipt verification
- **Support Us**: Donation page and fund transparency
- **Eco Assistant**: Chatbot answering environmental questions
- **Multilingual**: Full support for Azerbaijani and English languages

### Technologies

**Backend:**
- Express.js - Web server
- PostgreSQL - Database
- OpenAI API - Chatbot and receipt analysis

**Frontend:**
- React - User interface
- TypeScript - Type-safe JavaScript
- TanStack Query - Data management

**UI:**
- Tailwind CSS - Styling framework
- Shadcn UI - Component library
- Lucide React - Icon system
